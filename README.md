# dotfiles

This is  my set up for Arch Linux OS dotfiles.

Files included here are:

.zshrc
------
ZSH profile

.oh-my-zsh
----------
Custom themes for oh-my-zsh

.gitignore_global
-----------------
Files to be ignored system wide by Git history

.tmux.configure
---------------
Tmux tweaks like mouse functionality, scroll, menu bar etc.
